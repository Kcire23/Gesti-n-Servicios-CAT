# Gesti-n-Servicios-CAT
Sistema de Gestión de Solicitudes para el Centro de Atención Tecnológica (CAT)
Poker Planning:
Se utilizó Planning Poker para estimar el esfuerzo de cada Historia de Usuario, porque permite que todos los miembros del equipo aporten su criterio sin sesgos de opinión. Es una técnica ágil que promueve consenso, discusión y mejora la precisión de las estimaciones.
